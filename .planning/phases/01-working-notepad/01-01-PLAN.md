  <action>
Create the database singleton, pad CRUD operations, CSS design system, and utility functions.

**Database singleton (`src/lib/server/db.ts`):**
- Import `better-sqlite3`, create singleton instance
- Database path from `DB_PATH` env var, default to `./data/padplus.db`
- Create `data/` directory with `mkdirSync` if it doesn't exist
- Enable WAL mode: `db.pragma('journal_mode = WAL')`
- Enable synchronous NORMAL: `db.pragma('synchronous = NORMAL')`
- Schema migration using `PRAGMA user_version`:
  - Version 0 -> 1: Create `pads` table with columns: `id` (INTEGER PRIMARY KEY AUTOINCREMENT), `slug` (TEXT NOT NULL UNIQUE), `content` (TEXT NOT NULL DEFAULT ''), `version` (INTEGER NOT NULL DEFAULT 1), `collaboration_mode` (TEXT NOT NULL DEFAULT 'last-save-wins'), `created_at` (TEXT NOT NULL DEFAULT datetime('now')), `updated_at` (TEXT NOT NULL DEFAULT datetime('now'))
  - Create index on `slug`: `CREATE INDEX idx_pads_slug ON pads(slug)`
  - Set `PRAGMA user_version = 1`
- Export db as default

**Pad CRUD (`src/lib/server/pads.ts`):**
- Import db from `./db`
- Define TypeScript `Pad` interface: `{ id: number; slug: string; content: string; version: number; collaboration_mode: string; created_at: string; updated_at: string }`
- `getPadBySlug(slug: string): Pad | undefined` -- SELECT by slug
- `createPad(slug: string): Pad` -- INSERT with empty content, return the created pad
- `savePad(slug: string, content: string, expectedVersion: number): { conflict: boolean; pad: Pad }` -- UPDATE with `WHERE version = ?` check. If `result.changes === 0`, it's a conflict: return current pad with `conflict: true`. Otherwise return updated pad with `conflict: false`.
- Use prepared statements for performance (`.prepare()`)
- Validate slug: reject slugs starting with `_` or `api`, or containing file extensions (`.ico`, `.js`, `.css`, `.json`, `.map`, `.svg`, `.png`, `.jpg`). Throw an error for invalid slugs.

**Server hooks (`src/hooks.server.ts`):**
- Import `'$lib/server/db'` to trigger database initialization on server start
- Export an empty or minimal `handle` function (SvelteKit hook to ensure db import runs)
