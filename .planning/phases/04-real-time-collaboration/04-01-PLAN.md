1. Create `src/lib/server/ws-server.ts` -- core WebSocket handler shared by dev plugin:
   - Import: `* as Y from 'yjs'`, `* as syncProtocol from 'y-protocols/sync'`, `* as encoding from 'lib0/encoding'`, `* as decoding from 'lib0/decoding'`, `{ getYjsState, saveYjsState } from './yjs-persistence'`, `{ getPadBySlug } from './pads'`
   - Use `import type { WebSocket as WsWebSocket } from 'ws'` for typing
   - Define message types: `const messageSync = 0; const messageAwareness = 1;`
   - Room type: `{ doc: Y.Doc; clients: Set<WsWebSocket>; cleanupTimer: ReturnType<typeof setTimeout> | null; saveTimer: ReturnType<typeof setTimeout> | null }`
   - Export `const rooms = new Map<string, Room>()`
   - Export `function handleYjsConnection(ws: WsWebSocket, slug: string): void`:
     a. Get or create room synchronously (no async gap between check and set -- Research Pitfall 2)
     b. New room: create Y.Doc, load persisted state via getYjsState(slug). If state exists, Y.applyUpdate(doc, state). If no state but pad has content (getPadBySlug), seed Y.Text: `doc.getText('content').insert(0, pad.content)`. Cancel any pending cleanup timer.
     c. Register doc 'update' listener: debounce persistence at 500ms. On persist: `saveYjsState(slug, Buffer.from(Y.encodeStateAsUpdate(doc)), doc.getText('content').toString())`
     d. Add ws to room.clients
     e. Send sync step 1 to new client: `const encoder = encoding.createEncoder(); encoding.writeVarUint(encoder, messageSync); syncProtocol.writeSyncStep1(encoder, doc); ws.send(encoding.toUint8Array(encoder))`
     f. ws.on('message'): cast data to Uint8Array, decode messageType. For messageSync: create encoder, writeVarUint messageSync, readSyncMessage(decoder, encoder, doc, ws), if encoder has content send back to ws. Also broadcast the original message to all OTHER clients in room. For messageAwareness: ignore (no presence per user decision).
     g. ws.on('close'): remove ws from room.clients. If room is empty: persist final state immediately (cancel debounce, save now), then set 30-second cleanup timer. On timer: if still empty, doc.destroy(), rooms.delete(slug).
